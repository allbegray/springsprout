봄싹(SpringSprout)
===========================

## 개요
개발자 커뮤니티 [봄싹의 사이트](http://springsprout.org)를 메이븐, 스프링, 하이버네이트, 제이쿼리를 사용해서 개발하고 있습니다.
봄싹 개발에 참여해주셔서 감사합니다.

자세한 내용은 [봄싹 위키](http://wiki.springsprout.org/display/SSD/Home)를 참조해 주시기 바랍니다.
